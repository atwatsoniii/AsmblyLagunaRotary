# Plan Documents

This folder is the **canonical location for plan documents** in this workspace.

When creating or updating plans (e.g. with Cursor's plan feature), write plan markdown files here so they are:

- Visible in the project file tree
- Version-controlled with the rest of the repo

Use kebab-case filenames (e.g. `plans-in-workspace.md`, `safe-retracts-option.md`).
